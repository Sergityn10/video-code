Scroll-driven animations

El principal cambio para establecer animaciones mediante el scroll, es estableciendo la propiedad animation-timeline:scroll().

Posteriormente, es necesario, si se desea no ocupe todo el scroll, el animation-range, desde cuando empieza a cuando acaba.

Ademas, existen propiedades unicas de scroll, como las scroll-timeline, que es para establecer el timeline del scroll de dicho elemento y establecer en las animaciones de otro, para que dependan de ellas.
